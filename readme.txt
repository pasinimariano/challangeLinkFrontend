Para realizar el proyecto se utilizó:
    -React
    -React-router
    -Redux
    -Redux-persist
    -Redux-Thunk
    -Material Ui 
    -CSS
    -Axios

Todas las dependencias necesarias se encuentran detalladas en el archivo "package.json".
Utilizando el comando "npm i", se instalarán todas.
Notarás que se crea un directorio llamado "node_modules", esto indica que el comando anterior
funcionó correctamente.

Para correr el proyecto se debe poner el comando:
"npm start".
Este comando iniciará el servidor de webpack. Dentro de la configuración "webpack.config.js"
podrás cambiar el Port, pero por default el server correrá en el 3000.
